# 🎓 校园时光胶囊 - 前端项目 (Web01)

这是一个基于 Vue 3 和 Element Plus 构建的前端项目，用于实现校园时光胶囊的核心功能。

## ✨ 技术栈

- **前端框架**: Vue 3 (使用 Composition API)
- **构建工具**: Vite
- **UI 组件库**: Element Plus
- **路由管理**: Vue Router 4
- **状态管理**: (待定，小型项目可省略或用 Pinia)
- **网络请求**: Axios

## 🚀 运行项目

1.  **安装依赖**:
    ```bash
    npm install
    ```
2.  **启动开发服务器**:
    ```bash
    npm run dev
    ```
    项目将在 `http://localhost:5173/web01/` 启动。

3.  **构建生产版本**:
    ```bash
    npm run build
    ```
    构建后的文件将输出到 `dist` 目录。
